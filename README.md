## Getting Started

**1. Clone the repository:**

```
git clone https://github.com/TheIneq/OpenGL.git
```

**2. Build the project**

**You can build project using cmake:**

```
cmake . -B build -G Ninja && cmake --build build
```

**3. Run the project**

**On Linux**
```
./build/OGL
```

**On Windows**
```
build\OGL.exe
```
